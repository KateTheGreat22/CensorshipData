# Overview

This project helped me to better understand the bridge between front end and back end programming and data anaylsis. The purpose of the code is to go through the data from the American Library Association to see which book genres are currently being censored and which authors are consistently being censored. 

I obtained my data from Kaggle under the title "Banned Book Dataset"
https://www.kaggle.com/datasets/chielerli/banned-book-dataset 


The purpose behind this assignment is to help me better work on a personal research project on the impact of literary censorship. The data found here helped to streamline and refine my research.
